# Music Player🎵🎶 <a name="about-project"></a>

This is a simple Music Player project that allows you to play music. You can play, pause, and skip to the next song. You can also see the progress of the song and the duration of the song. This project is built with HTML, CSS, and JavaScript.

## 🛠 Built With <a name="built-with"></a>

- ✅ HTML
- ✅ CSS
- ✅ JavaScript
- ✅ Bootstrap
